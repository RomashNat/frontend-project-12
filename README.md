# Chat (Slack)
The real-time application for messaging between users

### Main technologies:
- React 19 - a library for building user interfaces
- Redux Toolkit - application state management
- React Router - client-side routing
- Vite - a modern frontend builder

### Installation:

Clone the repository:

```sh
$ git clone https://github.com/RomashNat/frontend-project-12.git
cd frontend-project-12
```

Install dependencies with npm install:

```sh
$ make install
```

Build the project with npm run build:

```sh
$ make build<img width="1915" height="869" alt="LoginPage" src="https://github.com/user-attachments/assets/caef42d0-daee-49aa-998b-2e6dc6db6f8e" />

```

Start the development server with npm start:

```sh
$ make start
```
For development:

```sh
$ cd frontend && npm run dev
```
